# Reader
一款简单的新闻阅读类APP，包含知乎日报、gank等内容，练手项目，希望有时间完善，非常感谢开源，设计思路主要借助于开源。

-------------

#### 所用到的开源技术
 - 主体框架 **MVP** ,减少 View 的逻辑代码
 - **Dragger2** 解耦,统一管理注入
 - 网络框架 **RxJava**，**Retrofit**，**okHttp**
 - 图片加载 **Glide**
 - 简约数据库 **Realm**
